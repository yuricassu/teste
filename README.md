# Documentador de Power BI

## Como Executar

### 1. Instalar dependências
```bash
pip3 install -r requirements.txt
```

### 2. Iniciar API
```bash
python3 api.py
```

### 3. Abrir frontend
Abra `index.html` no navegador

## Pronto!
Agora você pode fazer upload de arquivos .pbit e baixar a documentação em PDF.
